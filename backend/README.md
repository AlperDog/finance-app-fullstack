# Backend

Bu klasör, kişisel finans ve gider takip uygulamasının Node.js (Fastify) ile yazılmış backend API kodlarını içerir.

- Kimlik doğrulama (JWT)
- SQLite (Knex) ile veri yönetimi
- Redis ile cache
- Kullanıcı, işlem, kategori, bütçe ve analiz endpointleri

Tüm kodlar `src` klasöründe organize edilmiştir.
